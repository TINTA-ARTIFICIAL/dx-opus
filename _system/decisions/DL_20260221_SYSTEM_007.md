# DECISION LOG ENTRY: DL_20260221_SYSTEM_007

## DECISION
Ningún archivo del repositorio GitHub incluye versión en el nombre. Git gestiona el historial completo. La versión vive únicamente en la cabecera YAML del archivo y en el CHANGELOG interno.

## RATIONALE
Los nombres con versión (PROMPT_X_v1_3.md) generan deuda técnica: hay que renombrar el archivo en cada actualización, los enlaces se rompen, y la historia queda fragmentada. Git provee historial completo sin ese coste.

## AFFECTED SUBSYSTEMS
- Todos: la regla aplica a todos los archivos del repositorio sin excepción
