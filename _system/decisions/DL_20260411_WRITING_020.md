# DECISION LOG ENTRY: DL_20260411_WRITING_020

## DECISION

El modo híbrido de PROMPT_WRITE_POST — continuar texto parcialmente escrito por el editor — se activa solo por declaración explícita del editor. El sistema puede detectar automáticamente si hay texto con estructura de borrador en el input y pregunta al editor qué uso quiere hacer de él. Sin confirmación explícita del editor de que quiere completarlo, todo texto aportado se trata como material de notas y se procesa como tal desde PROMPT_POST_BRIEF.

## RATIONALE

El texto aportado por el editor puede tener múltiples roles: borrador a completar, material de referencia de estilo, draft descartado, o simplemente notas elaboradas. La presencia de texto con estructura de borrador no es señal suficiente para inferir el modo — el editor debe declarar explícitamente su intención. El comportamiento por defecto más conservador — tratar todo como notas — evita que el sistema modifique texto que el editor no quería que se tocara.

## AFFECTED SUBSYSTEMS

- WRITING: PROMPT_POST_BRIEF detecta texto y pregunta; PROMPT_WRITE_POST activa modo híbrido solo con confirmación

## ARTIFACTS AFFECTED

| Artifact | Action | Notes |
|---|---|---|
| PROMPT_POST_BRIEF | CREATE v1.0 | Detecta texto con estructura de borrador y pregunta por su rol antes de procesarlo |
| PROMPT_WRITE_POST | MODIFY v1.0 → v2.0 | Modo híbrido activado solo por declaración explícita del editor recibida desde PROMPT_POST_BRIEF |
