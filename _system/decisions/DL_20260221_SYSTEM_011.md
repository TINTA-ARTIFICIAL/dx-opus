# DECISION LOG ENTRY: DL_20260221_SYSTEM_011

## DECISION
Las herramientas operativas (TOOLING) pertenecen al subsistema SYSTEM mientras sean menos de 3 activas. Si superan ese número se extraen a un subsistema TOOLING independiente. El tipo de artefacto para herramientas es TOOL.

## RATIONALE
Evitar crear un subsistema para 1-2 herramientas. SYSTEM puede gestionar herramientas de forma natural hasta que su número justifique la extracción. El umbral de 3 es pragmático y revisable.

## AFFECTED SUBSYSTEMS
- SYSTEM: owner de TOOL_SETUP_PROJECT y TOOL_GITHUB_REPO_STRUCTURE mientras sean menos de 3 herramientas

|---|---|---|

| TOOL_GITHUB_REPO_STRUCTURE | CREATE v1.0 | Segunda herramienta operativa. Creada en Fase 0 |
